# NBN EN 1997-2:2007 — Annexe A Liste des résultats d'essai

## Annexe A (informative) — Liste des résultats d'essai des normes d'essais géotechniques

(1) Le Tableau A.1 récapitule les essais de laboratoire et en place avec, pour chacun d'eux, les résultats des essais qu'il convient de faire figurer dans le rapport de reconnaissance géotechnique (s'il y a lieu).

### Tableau A.1 — Liste des résultats des essais d'après les normes géotechniques

#### Essais en place (voir Section 4)

| Essai | Résultats d'essai à consigner |
|---|---|
| CPT — Essai de pénétration statique au cône | résistance à la pénétration du cône ($q_c$) ; frottement latéral unitaire local ($f_s$) ; rapport de frottement ($R_f$) |
| CPTU — Essai de pénétration statique au piézocône | résistance à la pénétration du cône corrigée ($q_t$) ; frottement latéral unitaire local ($f_s$) ; pression interstitielle ($u$) |
| DP — Essai de pénétration dynamique | nombre de coups $N_{10}$ (DPL, DPM, DPH) ; nombre de coups $N_{10}$ ou $N_{20}$ pour DPSH |
| SPT — Essai de pénétration au carottier | nombre de coups $N$ ; rapport d'énergie $E_r$ ; description du sol |
| PMT Ménard (MPM) | module pressiométrique ($E_M$) ; pression de fluage ($p_f$) ; pression limite ($p_{LM}$) ; courbe d'expansion |
| FDT — Essai au dilatomètre flexible (roche) | module dilatométrique ($E_{FDT}$) ; courbe de déformation |
| Autres essais pressiométriques (PBP, SBP, FDP) | courbe de dilatation |
| FVT — Essai au scissomètre de chantier | résistance au cisaillement non drainée non corrigée ($c_{fv}$) ; résistance remaniée ($c_{rv}$) ; courbe couple-rotation |
| WST — Essai de sondage par poids | enregistrement continu : profondeur de pénétration pour charge normalisée OU nombre de demi-tours pour enfoncer la sonde de 0,2 m sous charge normalisée de 1 kN |
| PLT — Essai de chargement à la plaque | pression de contact ultime ($p_u$) |
| DMT — Essai au dilatomètre plat | pression corrigée ($p_0$) ; pression de dilatation corrigée ($p_1$) à 1,1 mm ; module dilatométrique $E_{DMT}$ ; indice de matériau ($I_{DMT}$) ; indice lié à la contrainte horizontale ($K_{DMT}$) |

#### Essais de laboratoire (voir Section 5)

| Essai | Résultats d'essai à consigner |
|---|---|
| Teneur en eau (sol) | valeur de la teneur en eau ($w$) |
| Masse volumique (sol) | valeur de la masse volumique ($\rho$) |
| Masse volumique des particules (sol) | valeur de la masse volumique des particules ($\rho_s$) |
| Granularité (sol) | courbe granulométrique |
| Limites de consistance (sol) | $w_L$ (limite de liquidité) et $w_P$ (limite de plasticité) |
| Indice de densité (sol) | $e_{max}$, $e_{min}$, indice de densité $I_D$ |
| Teneur en matières organiques (sol) | $C_{OM}$ |
| Teneur en carbonates (sol) | $C_{CaCO_3}$ |
| Teneur en sulfates (sol) | $C_{SO_4^{2-}}$ ou $C_{SO_3^{2-}}$ |
| Teneur en chlorure (sol) | $C_{Cl}$ |
| pH (sol) | valeur du pH |
| Compressibilité à l'œdomètre (sol) | courbe de compressibilité ; courbes de consolidation ; courbe de compression secondaire (fluage) ; $E_{oed}$ (avec intervalles de contraintes) et $\sigma'_p$ ; ou $[C_s, C_c]$ ; $C_\alpha$ |
| Scissomètre de laboratoire (sol) | indice de résistance ($c_u$) |
| Cône tombant en chute libre (sol) | indice de résistance ($c_u$) |
| Compression simple (sol) | indice de résistance $q_u = 2 c_u$ |
| Compression non consolidée non drainée — UU (sol) | résistance au cisaillement non drainé ($c_u$) |
| Compression triaxiale consolidée — CD ou CU (sol) | courbes contrainte-déformation et de pression interstitielle ; chemin de contrainte ; cercles de Mohr ; $c'$, $\varphi'$ ou $c_u$ ; variations de $c_u$ avec $\sigma'_c$ ; paramètre de déformation ($E'$) ou ($E_u$) |
| Cisaillement consolidé à la boîte (sol) | courbe contrainte-déplacement ; diagramme $\tau$–$\sigma$ ; $c'$, $\varphi'$ ; paramètres résiduels |
| Portance CBR (sol) | indice CBR ($I_{CBR}$) |
| Perméabilité (sol) | coefficient de perméabilité ($k$) à partir : de l'essai de perméabilité direct en laboratoire ; des essais de perméabilité en place ; de l'essai œdométrique |
| Teneur en eau (roche) | valeur de la teneur en eau $w$ |
| Masse volumique et porosité (roche) | valeur de la masse volumique $\rho$ et de la porosité $n$ |
| Gonflement (roche) | indice de gonflement (déformation) ; pression de gonflement ; gonflement libre ; gonflement sous charge constante |
| Compression uniaxiale et déformation (roche) | $\sigma_c$ ; module de déformation ($E$) ; coefficient de Poisson ($\nu$) |
| Essai de charge ponctuelle (roche) | indice de résistance $I_{s,50}$ |
| Essai de cisaillement direct (roche) | courbe contrainte-déplacement ; diagramme de Mohr ; $c'$, $\varphi'$ ; paramètres résiduels |
| Essai brésilien (roche) | contrainte de traction ($\sigma_T$) |
| Essai de compression au triaxial (roche) | courbe(s) contrainte-déformation ; chemins de contrainte ; cercles de Mohr ; $c'$, $\varphi'$ ; module de déformation ($E$) et coefficient de Poisson ($\nu$) |
