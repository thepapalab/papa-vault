# PTR CE01 Fascicule 1 — Index

Infrabel's primary technical prescription for the construction of civil engineering structures and buildings. Supplements or replaces corresponding Eurocode prescriptions. Applies to all structures managed by Infrabel and any works that could affect track stability.

> [!important] Governing framework
> PTR CE01 is applied as a document attached to the Eurocodes. Where PTR CE01 is silent, the relevant NBN EN applies with its Belgian National Annex. Where PTR CE01 prescribes, it takes precedence over the Eurocode.
> Consequence class CC2 (NBN EN 1990) applies to all civil engineering structures unless otherwise specified in the special CSC.

## Hard constraints imposed by PTR CE01

| Parameter | Value | Clause |
|-----------|-------|--------|
| **f_yk** for passive reinforcement | ≤ **400 N/mm²** (BE500 treated as BE400 for calculations) | §5.4.1 |
| Reinforcement type | B500B only (not B500A / DE500 BS) | §5.4.1 |
| Min. concrete class (bridges) | C30/37 | §5.3.1 |
| Max. concrete class (bridges) | C90/105 | §5.3.1 |
| LM71 classification factor α | **1.20** (ULS, elastic design, deflection) | §3.1.6.3 |
| γ_c (durable + transient) | 1.50 | §5.2.2 |
| γ_s (durable + transient) | 1.15 | §5.2.2 |
| Crack width limit (submerged / groundwater contact) | 0.20 mm | §5.8.4 |

> [!note] α_cc not set by PTR CE01
> PTR CE01 Ch5 defers to NBN EN 1992-1-1 in full ("Pour autant qu'il n'y soit pas dérogé..."). α_cc is a Nationally Determined Parameter governed by the **NBN EN 1992-1-1 ANB** — PTR CE01 imposes no override. See [[EC2 — ULS material design values]] for the open flag.
