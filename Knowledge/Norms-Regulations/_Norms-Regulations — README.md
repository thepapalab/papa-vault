# Norms-Regulations — README

The *map* to normative sources — not a restatement of method. For a given check: which standard, which clause, which National Annex, which Infrabel rule, and **where the PDF lives** (path or location). The method itself stays in `Methodology/`; the formulas in `Formulas/`.

A typical entry records: standard reference and title; the clause/table that matters; the Nationally Determined Parameters and their Belgian ANB values; the applicable Infrabel PTR article; and the file location of the authoritative PDF.

## Subfolders

- **Eurocodes-NBN/** — EN 199x series + Belgian NBN ANB annexes. NDP values live here.
- **Infrabel-PTR/** — Infrabel technical prescriptions. PTR CE01 (fyk ≤ 400 MPa) and related CE-series.
