# Worked-Examples — README

Good solved reference cases worth emulating. Each note: one example, clearly **dated** and **sourced** (which project, which textbook, which validated notebook run). These are templates of *good output*, not the general method.

Distinct from:
- **Methodology/** — the general method (a worked example is one instance of it).
- **Salvage/** — legacy files awaiting conversion (a worked example is finished and trusted).

Good candidates: a validated calc-note output (e.g. the 521.1 mm settlement reference), a cleanly-structured concept note, a well-formed contractor reply, a hypothesis note that aged well.
