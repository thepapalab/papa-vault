## Βασικά Μεγέθη & Τιμές (Γρήγορη Αναφορά)

### Επιβαλλόμενα φορτία (EN 1991-1-1)
| Κατηγορία | qk (kN/m²) | Qk (kN) |
|---|---|---|
| A (κατοικίες) | 2,0 | 2,0 |
| C1 (με τραπέζια) | 3,0 | 3,0 |
| C2–C4 | 5,0 | 4,0 |
| C5 (συναυλίες) | 7,5 | 4,5 |
| F (≤30 kN οχήματα) | 2,5 | 20 |
| G (30–160 kN) | 5,0 | 90 |
| H (στέγη) | 0,5 | 1,0 |

### Φορτία χιονιού (EN 1991-1-3)
| Ζώνη | sk,0 | Τυπικές περιοχές |
|---|---|---|
| Α | 0,40 kPa | Νησιά, Πελοπόννησος |
| Β | 0,80 kPa | Υπόλοιπη Ελλάδα |
| Γ | 1,70 kPa | Θεσσαλία, Σποράδες, Εύβοια |

### Άνεμος (EN 1991-1-4)
- Νησιά / ακτή ≤10 km: vb,0 = **33 m/s** → qb = 0,681 kN/m²
- Υπόλοιπη χώρα: vb,0 = **27 m/s** → qb = 0,455 kN/m²

### Θερμικές (EN 1991-1-5)
- Tin: Καλοκαίρι 20°C / Χειμώνας 25°C
- Tout (Ν-ΔΩ, α=0,9): Tmax+42°C (καλοκαίρι), Tmin (χειμώνας)
- Υπόγεια >1m: ±5°C (χειμ. −3°C / καλοκ. +5°C)

### Πυρκαγιά (EN 1991-1-2 via Vassart)
- Efi,d = G + ψ2·Q (accidental combination)
- ψ2 = 0,3 (offices), 0,6 (shopping/congregation), 0,8 (storage)
- ISO 834: θ = 20 + 345·log(8t+1) [°C]
