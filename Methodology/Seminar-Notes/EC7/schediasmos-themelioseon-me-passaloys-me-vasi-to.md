# Σχεδιασμός Θεμελιώσεων με Πάσσαλους (EC7.1)

**Πηγή:** Παπαδόπουλος Β., Τομέας Γεωτεχνικής ΕΜΠ · 45 σελ.

Πλήρης εφαρμογή του ΕΝ 1997-1 στον σχεδιασμό πασσάλων: αξονική θλίψη, εφελκυσμός, εγκάρσια φόρτιση, και SLS — με επτά αριθμητικά παραδείγματα (C-1 ÷ C-4, T-1, TR-1, TR-2).

---

## 1. Οριακές Καταστάσεις Πασσάλων

### 1.1 ULS

| Τύπος | Περιγραφή |
|-------|-----------|
| **GEO** | Αστοχία εδάφους: υψηλές θλιπτικές φορτίσεις, εγκάρσια φόρτιση, εξόλκευση, απώλεια ολικής ευστάθειας |
| **STR** | Δομική αστοχία πασσάλου (θλίψη, εφελκυσμός, κάμψη) |

Βασική ανίσωση: $E_d \leq R_d$

### 1.2 SLS

- Υπερβολικές καθιζήσεις
- Υπερβολικές ανυψώσεις
- Πλευρικές μετατοπίσεις πέραν ανεκτών ορίων
- Μη αποδεκτές ταλαντώσεις

Ελέγχεται με $E_d \leq C_d$ και $\gamma = 1$ για όλους τους επιμέρους συντελεστές.

## 3. Αξονική Θλιπτική Φόρτιση

### 3.1 Βασική Ανίσωση

$$F_{c,d} \leq R_{c,d} = \frac{R_{c,k}}{\gamma_R}$$

Η $R_{c,k}$ (χαρακτηριστική αντοχή = οριακό φορτίο) υπολογίζεται με 3 εναλλακτικούς τρόπους.

### 3.2 Μέθοδος 1: Από Στατικές Δοκιμαστικές Φορτίσεις

$$R_{c,k} = \min\!\left\{\frac{(R_{c,m})_\mathrm{mean}}{\xi_1},\ \frac{(R_{c,m})_\mathrm{min}}{\xi_2}\right\}$$

**Πίνακας ξ (Πίν. Α.9):**

| n | ξ1 | ξ2 |
|---|----|----|
| 1 | 1.40 | 1.40 |
| 2 | 1.30 | 1.20 |
| 3 | 1.20 | 1.05 |
| 4 | 1.10 | 1.00 |
| ≥5 | 1.00 | 1.00 |

> Για κεφαλόδεσμο με επαρκή ακαμψία (ανακατανομή φορτίων): μείωση ξ κατά 10% (προκύπτει ξ_eff = ξ/1.10).

**Παράδειγμα C-1** — Πάσσαλοι εκτοπίσεως Φ0.40m (n=2 δοκιμές):
- Rc,m,1 = 5.00 MN, Rc,m,2 = 5.60 MN → mean=5.30, min=5.00
- Rc,k = min{5.30/1.30; 5.00/1.20} = min{4.077; 4.167} = **4.077 MN**
- Rc,d = 4.077/1.10 = **3.706 MN** (γt=1.10, πάσσαλοι εκσκαφής)
- ΣFc,d = 1.35×20 + 1.50×5 = 34.50 MN → απαιτούνται **10 πάσσαλοι** (ή 9 με ακαμπτό κεφαλόδεσμο)

### 3.3 Μέθοδος 2: Από Γεωτεχνικές Δοκιμές (Πενετρομετρήσεις/Γεωτρήσεις)

$$R_{c,k} = \min\!\left\{\frac{(R_{c,\mathrm{cal}})_\mathrm{mean}}{\xi_3},\ \frac{(R_{c,\mathrm{cal}})_\mathrm{min}}{\xi_4}\right\}$$

όπου $R_{c,\mathrm{cal}} = R_{b,\mathrm{cal}} + R_{s,\mathrm{cal}}$ υπολογισμένα από τα αποτελέσματα n επιτόπου δοκιμών.

**Πίνακας ξ3, ξ4 (Πίν. Α.10):**

| n | ξ3 | ξ4 |
|---|----|----|
| 1 | 1.40 | 1.40 |
| 2 | 1.35 | 1.27 |
| 3 | 1.33 | 1.23 |
| 4 | 1.31 | 1.20 |
| 5 | 1.29 | 1.15 |
| 7 | 1.27 | 1.12 |
| 10 | 1.25 | 1.08 |

**Εναλλακτική μέθοδος (χαρακτηριστικές τιμές μοναδιαίων αντιστάσεων):**

$$R_{b,k} = A_b \cdot q_{b,k}, \qquad R_{s,k} = \sum A_{s,i} \cdot q_{s,i,k}$$

$$R_{c,d} = \frac{R_{b,k}}{\gamma_b \cdot \gamma_m} + \frac{R_{s,k}}{\gamma_s \cdot \gamma_m}$$

**Συντελεστής προσομοίωσης γm:**

| Τρόπος υπολογισμού qb,k, qs,k | γm |
|-------------------------------|-----|
| Μέσω ξ (π.χ. από SPT, CPT) — τιμές ξ ενσωματώνουν αβεβαιότητα | 1.00 |
| Από εργαστηριακές/επιτόπου δοκιμές χωρίς συντελεστές ξ | **1.30** |

**Παράδειγμα C-2** — Πάσσαλος εκτοπίσεως Φ0.50m, L=11m (n=3 CPT):
- (Rc,cal)mean = 2645 kN, (Rc,cal)min = 2239 kN
- n=3: ξ3=1.33, ξ4=1.23 → Rc,k = min{2645/1.33; 2239/1.23} = min{1989; 1820} = **1820 kN**
- Rc,d = 1820/1.10 = **1645 kN**

**Παράδειγμα C-3** — Πάσσαλος εκσκαφής Φ0.60m (DIN 4014, γm=1.30):
- qc=0.4×N=10 MPa → qb,k=2.00 MPa, qs,k=0.08 MPa
- Rb,k = (π×0.60²/4)×2000 = 565 kN
- Rs,k = π×0.60×L×80 = 150.8·L (kN)
- Fc,d = 1.35×1200 + 1.50×200 = 1920 kN
- Rd = 565/(1.10×1.30) + 150.8·L/(1.10×1.30) = 395 + 105.4·L ≥ 1920 → **L ≥ 14.5 m**

### 3.4 Μέθοδος 3: Από Δοκιμές Δυναμικής Εμπήξεως

$$R_{c,k} = \min\!\left\{\frac{(R_{c,m})_\mathrm{mean}}{\xi_5},\ \frac{(R_{c,m})_\mathrm{min}}{\xi_6}\right\}$$

**Πίνακας ξ5, ξ6 (Πίν. Α.11):**

| n | ξ5 | ξ6 |
|---|----|----|
| ≥2 | 1.60 | 1.50 |
| ≥5 | 1.50 | 1.35 |
| ≥10 | 1.45 | 1.30 |
| ≥15 | 1.42 | 1.25 |
| ≥20 | 1.40 | 1.25 |

> Οι ξ5, ξ6 πολλαπλασιάζονται επιπλέον με κατάλληλο γm αναλόγως τύπου μετρήσεων και αξιολόγησης.

## 4. Αρνητικές Τριβές (Negative Skin Friction)

**Παράδειγμα C-4** — Πάσσαλος εκτοπίσεως Φ0.30m, Gk=300 kN, επιφόρτιση q=40 kPa:

Μέθοδος β: $\beta = 0.25 \div 0.40$, τιμή $q'_{D,k} = \beta \cdot (\gamma' z + q)$

Για β=0.30, LD=3.5m: $q'_{D,k} = (7.5 \times 3.5 + 40) \times 0.30 = 20$ kPa

$$F_{K,D} = \pi B L_D q'_{D,k} = 132\ \text{kN} \quad\text{(αρνητικές τριβές)}$$

$$R_{s,k} = \pi B L_R q'_{s,k} = 56.55 L_R\ \text{kN}, \quad R_{b,k} = 64\ \text{kN}$$

Φορτίο σχεδιασμού: $F_{c,d} = 1.35 G_k + 1.35 F_{D,k} = 583$ kN

$$R_{c,d} = \frac{R_{b,k} + R_{s,k}}{1.10} = 58.2 + 51.41\,L_R \geq 583 \implies L_R = 10.2\ \text{m}$$

Συνολικό μήκος: $L = L_D + L_R = 3.5 + 10.2 = 17.2$ m

## 5. Αξονική Εφελκυστική Φόρτιση

$$F_{t,d} \leq R_{t,d} = \frac{R_{t,k}}{\gamma_{s,t}}$$

Η $R_{t,k}$ εκτιμάται αντίστοιχα με τις θλιπτικές φορτίσεις. Λαμβάνεται υπόψη **μόνο** η πλευρική τριβή ($R_{b,k} = 0$).

**Παράδειγμα T-1** — Πάσσαλος εκσκαφής Φ0.80m, μεταβλητό εφελκυστικό Ft=1040 kN:

Εδαφικό προφίλ: Στρώση Ι (0–5 m, qs,k=50 kPa), Στρώση ΙΙ (5–(5+l₂) m, qs,k=100 kPa)

$$F_{t,d} = 1.50 \times 1040 - 1.0 \times G_K = 1497.2 - 12.56\,l_2\ \text{(kN)}$$

$$R_{t,k} = \pi \times 0.8 \times (5 \times 50 + l_2 \times 100) = 628.3 + 251.3\,l_2\ \text{(kN)}$$

$$R_{t,d} = \frac{R_{t,k}}{\gamma_{s,t} \cdot \gamma_m} = \frac{R_{t,k}}{1.15 \times 1.30} = 420.3 + 168.1\,l_2$$

Από $F_{t,d} \leq R_{t,d}$: $l_2 \geq 5.96$ m → **L = 5.0 + 6.0 = 11.0 m**

## 6. Εγκάρσια (Οριζόντια) Φόρτιση

$$F_{tr,d} \leq R_{tr,d}$$

### 6.1 «Κοντός» Πάσσαλος — Μηχανισμός GEO

Αστοχία εδάφους: μετατοπίσεις ή στροφή κεφαλής.

**Παράδειγμα TR-1** — Πάσσαλος εκσκαφής Φ1.0m, cu=50 kPa, Ftr,k=300 kN, RM,d=2400 kNm:

Με Broms:
$$f = \frac{H_u}{9 c_u B} = \frac{H_u}{450}, \qquad M_{\max} = H_u\bigl(e + 1.5B + 0.5f\bigr) = 2.25\,g^2\,c_u B$$

Επίλυση: $R_{tr,k} = 609$ kN, $M_k = 1935$ kNm

$$R_{tr,d} = \frac{609}{\gamma_R \cdot \gamma_m} = \frac{609}{1.40 \times 1.00} = 435\ \text{kN}$$

$$F_{tr,d} = 1.35 \times 300 = 405\ \text{kN} \leq 435\ \text{kN} \checkmark$$

Και $M_d < M_k = 1935 < R_{M,d} = 2400$ kNm → ο πάσσαλος επαρκεί.

### 6.2 «Μακρύς» Πάσσαλος — Μηχανισμός STR

Αστοχία πασσάλου σε κάμψη (+ ενδεχομένως τοπική διατμητική αστοχία εδάφους στην κεφαλή).

**Παράδειγμα TR-2** — Πάσσαλος εκσκαφής Φ1.0m, cu=50 kPa, E=15 MPa, Ftr,k=700 kN, RM,d=2400 kNm:

Ο EC-7 συνιστά **προσομοίωμα Winkler** (δείκτης Kh). Εδώ λύθηκε με FEM 3D.

**DA-2 (γF επί δράσεων):** Ftr,d = 1.35×700 = 945 kN → Md = 2400 kNm = RM,d → οριακώς επαρκεί.

**DA-2\* (γE επί αποτελέσματος):** Frep = 700 kN → Mk = 1600 kNm → Md = 1.35×1600 = 2160 kNm < 2400 kNm

Παρατηρείται **διαφορά ~10% μεταξύ DA-2 και DA-2\*** στην εγκάρσια φόρτιση λόγω του μη-γραμμικού αποτελέσματος (ροπή κάμψεως ≠ γραμμική συνάρτηση φορτίου).

## 7. Οριακές Καταστάσεις Λειτουργικότητας (SLS)

$$E_d \leq C_d, \qquad \gamma = 1\ (\text{για δράσεις και εδαφικές ιδιότητες})$$

Σε απλές περιπτώσεις: αρκεί η διαπίστωση ότι ενεργοποιείται **μέρος μόνο** της αντοχής (το περιθώριο ULS συνήθως καλύπτει και τις SLS απαιτήσεις — ισχύει για εδάφη μέσης/υψηλής πυκνότητας).

**Κανονικοποιημένο διάγραμμα Rmob/Rult — s/B:**

Για πάσσαλους εκσκαφής (δεδομένα Ελλάδας): συνολικός συντελεστής $\gamma = \gamma_F \cdot \gamma_t \cdot \xi = 1.70 \div 2.15$ αντιστοιχεί σε $R_{mob}/R_{ult} \approx 0.50 \div 0.60$ στην κανονική φόρτιση → καθιζήσεις εντός αποδεκτών ορίων.
